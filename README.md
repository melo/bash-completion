**NAME**

Bash::Completion - Extensible system to provide bash completion

**SYNOPSIS**

~~~ bash
## For end users, in your .bashrc:
##
. setup-bash-complete
##
## Now install all the Bash::Completion::Plugins:: that you need
##
## For plugin writters, see Bash::Completion::Plugin
~~~
